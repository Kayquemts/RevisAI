# Technology Stack - RevisAI

| Category | Technology | Details |
| :--- | :--- | :--- |
| **Languages** | JavaScript (ESM/CommonJS) | Node.js no backend, React no frontend |
| **Frontend Framework** | React 19 | Single Page Application (SPA) |
| **Build Tool** | Vite 8 | Desenvolvimento rápido e build otimizado |
| **Styling** | Tailwind CSS 4 | Framework CSS utilitário |
| **Animations** | GSAP 3 | Biblioteca para animações avançadas |
| **Icons** | Lucide React | Conjunto de ícones consistentes |
| **Routing** | React Router Dom 7 | Gerenciamento de rotas no frontend |
| **Backend Framework** | Express 5 | Servidor Node.js minimalista |
| **Database** | Firebase Firestore | Banco de dados NoSQL em tempo real |
| **AI Providers** | OpenAI, AWS Bedrock, AWS SageMaker | Abstração para múltiplos provedores de LLM |
| **Environment** | Dotenv | Gerenciamento de variáveis de ambiente |

## Runtime Environment
- **Node.js**: Requerido para backend e build do frontend.
- **Package Manager**: npm (presença de `package-lock.json`).

## Key Dependencies
- `express`: Servidor web.
- `firebase-admin`: Integração com serviços Google Cloud/Firebase.
- `openai`: Cliente oficial para GPT.
- `aws-sdk`: Integração com serviços AWS.
- `gsap`: Utilizado para experiências visuais premium no frontend.
